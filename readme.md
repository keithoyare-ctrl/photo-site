## Features
- Responsive grid gallery
- Lightbox preview
- Project pages with metadata (camera, lens, location, story)
- Easy-to-update assets folder

## Usage
1. Clone the repo:
    git clone https://github.com/yourusername/photography-portfolio.git
2. Put optimized images in `assets/`
3. Edit captions and project metadata in the Markdown or JSON files used by the site
4. Open `index.html` in a browser or deploy to GitHub Pages / Netlify

## Suggested Project Structure
- assets/               ← photos and thumbnails
- css/                  ← styles
- js/                   ← lightbox and gallery scripts
- projects/             ← per-project markdown or JSON
- index.html
- README.md

## Tips
- Use web-friendly sizes (1200–2400px long edge) and thumbnails (300–600px).
- Include EXIF-free exports for privacy when sharing online.
